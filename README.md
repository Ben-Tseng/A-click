# A-click

Demo：https://github.com/Ben-Tseng/A-click/blob/main/Demo%20for%20A_click.gif

Chrome Web Store：https://chromewebstore.google.com/detail/auto-select-first/pngimbifkcibcinlegmnehcajnhhajkm

Firefox AMO: https://addons.mozilla.org/addon/a-click/

Chrome extension to automatically select all "Yes" options on web pages with one click. Features smart detection, iframe support, and visual feedback.
